# Beyond B — Platform Overview
## Gamified University Application Platform | MG5661 Item 2 | 🏆 Distinction

---

## Concept Summary

Beyond B is a behaviourally informed, gamified web platform that reimagines
the university application journey for Gen Z students. Built as a consultancy
prototype for B University, it addresses two systemic failures in the current
UCAS-based process: low engagement and low decision-making confidence.

---

## Platform Architecture

### Student Layer
| Feature | Purpose | Behavioural Principle |
|---|---|---|
| Quest Map | Gamified step-by-step application journey | Reduces cognitive load |
| Career Trait Quiz | Personalised course and career matching | Informed decision-making |
| Mini-Games | Simulate career day-in-the-life scenarios | Exploration-based learning |
| Avatar Builder | Custom character creation | Identity expression |
| Virtual Campus Tour | Interactive campus exploration | Reduces geographic barriers |
| Faculty Introductions | Meet department lecturers | Builds institutional trust |

### Admissions Layer
| Feature | Purpose |
|---|---|
| BI Dashboard | Real-time KPIs — applications, acceptance rate, engagement mix |
| Weekly Quiz Analytics | Track student engagement trends |
| Course Save Tracking | Monitor student interest patterns |
| Export Function | Download reports for admissions team |

## Application Journey — Quest Map

| Stage | Quest Name | Key Actions | XP Available |
|---|---|---|---|
| 1 | 🏘️ Application Village | Register, Career Quiz, Mini-Games, Choose Course | 300 XP |
| 2 | 🏰 Admissions Fortress | Interviews, Conditional Offer, Unconditional Offer | 300 XP |
| 3 | ✅ Acceptance Castle | Accept Offer, Pay Deposit, Accommodation | 300 XP |
| 4 | 🏔️ Preparation Peak | Finance, CAS, Visa, Professional Checks | 300 XP |
| 5 | 🎓 Enrolment Gateway | eVision Registration, Tuition Payment | 300 XP |

**Total XP Target: 1,500 XP across full journey**

## Why Gen Z Needs a New Approach

| What's Broken | What Gen Z Expects | How Beyond B Solves It |
|---|---|---|
| Static & stressful forms | Interactive & visual | Gamified Quest Map + Mini-Games |
| No personalisation | Content adapting to goals | Career Trait Quiz + matching |
| 79% need personal statement help | Guided structured support | Step-by-step Quest journeys |
| One-size-fits-all journey | Creative self-expression | Avatar Builder |
| Long waits, no feedback | Real-time progress | XP system + BI dashboard |

## Mini-Games — Career Exploration

### Game 1: A Day in the Life — Marketing Consultant
- Scenario-based decision game simulating real professional challenges
- Tracks: Strategy Points, Client Relations, Creativity
- Students experience the role before committing to a course

### Game 2: BI Quest Adventure
- 5 Epic Quests exploring Business Intelligence concepts and tools
- Legendary rewards + 250+ XP earnable
- Designed to help students discover aptitude for analytical roles

## Avatar System

Students build a personalised character selecting from:
- Complexion, hair styles, eye colour
- Professional outfit (Accountant, Doctor, Consultant, Engineer, Teacher)
- Accessories — bag, shoes

The avatar evolves as the student progresses through quests,
reinforcing identity and emotional investment in the journey.

## Ethical Considerations

| Risk | Mitigation |
|---|---|
| Predictive analytics bias | Trait mapping must not disadvantage underrepresented groups |
| Nudging transparency | Students must understand how recommendations are generated |
| Data usage clarity | Explicit plain-language consent and data retention policies |
| Student autonomy | Platform guides, never directs — final decisions remain with student |
